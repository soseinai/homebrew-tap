# Sosein Homebrew Tap

A [Homebrew](https://brew.sh) tap for [soseinai](https://sosein.ai) command-line tools.

## Usage

```sh
brew install soseinai/tap/ought
```

Or, equivalently:

```sh
brew tap soseinai/tap
brew install ought
```

## Available formulae

- **[ought](https://github.com/soseinai/ought)** — Behavioral test framework powered by LLMs

## How this tap is maintained

Formula updates are pushed automatically by the `sosein-release-bot` GitHub App
whenever a new release is cut from [`soseinai/ought`](https://github.com/soseinai/ought).
You generally shouldn't need to edit anything here by hand.
